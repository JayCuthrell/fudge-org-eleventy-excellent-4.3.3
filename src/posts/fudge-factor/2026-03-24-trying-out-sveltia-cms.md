I considered using [Decap CMS](https://decapcms.org/) for a while, but never got around to establishing the CMS authentication service to remove dependencies from Netlify and Cloudflare (worker).

At this point, adding a single HTML page with only a GitHub dependency is an acceptable alternative. One fine grained personal access token later, and I am speaking into my transcribing laptop with great success.

If you're reading this, then you will know that [Sveltia CMS](https://sveltiacms.app/en/docs/backends/github) is not just working... it is working well.
